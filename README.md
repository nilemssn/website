# Nile MSSN Website

Here's the source code repo for the Nile MSSN website. (Work currently in progress).

## Set up locally
1. Make sure you have the [Jekyll requirements](https://jekyllrb.com/docs/installation/) installed locally.
2. Install Jekyll and Bundler, if you don't already have them, with `gem install jekyll bundler`.
3. Clone this repo.
4. Run `bundle exec jekyll serve` to start the development server.
5. Go to `localhost:4000` on your browser to view the website.
6. You can start making changes to the site now.
